## 🔍 Troubleshooting Performance

If you notice stuttering during movement, verify these settings:

*   **Graphics Driver:** Ensure you use the current driver provided by NVIDIA or AMD. 
*   **Power Settings:** Set your Windows machine to High Performance mode within the Control Panel. 
*   **Background Tasks:** Close web browsers or resource-heavy applications before playing. This frees up memory for the game engine.
*   **Shader Cache:** If textures appear glitchy, delete the contents of the Cache folder inside the application directory. The game rebuilds these files upon the next launch.
